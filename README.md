## Install
[Node.js](https://nodejs.org)
[EJS](https://ejs.co)


## Requirements

 npm start to exec it, need to install nodemon and ejs
 ```sh
npm i
```


## Run tests

Test it executing this command on terminal:
```sh
npm start
```
And open [Localhost](http://localhost:3000)
